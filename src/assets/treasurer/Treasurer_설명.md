# Treasurer 설명

# Home

- 파일명: Home
- 버전: v.1.16.3
- 화면설명: 홈 메인 화면
1. 이벤트 배너
    - 운영팀에서 배너 콘텐츠 관리
    - 클릭 시 이벤트 상세 화면으로 이동
2. 내 자산
    - 사용자의 총 평가 자산 금액 노출
    - 거래하기 → 원자재 메뉴 드롭다운 노출
    - 정기구독 → [정기구독] 화면으로 이동
    - 입금하기 → 예치금 입금 바텀시트 노출
    - 더보기 → 출금하기/포인트내역 메뉴 드롭다운 노출
3. 실물 금 입금
    - [실물 금 입금] 화면으로 이동
4. 원자재 시세
    - 원자재 시세 실시간 노출
    - 클릭 시 각 원자재 상세 화면으로 이동
5. 데일리 1분 퀴즈
    - 하루 최대 2개의 퀴즈 참여 가능
    - 퀴즈 완료 시 1포인트 지급
6. 지식 모으기
    - 클릭 시 [경제 레슨] 화면으로 이동
7. 출석체크
    - 클릭 시 [출석체크] 화면으로 이동
8. 주요지수
    - 주요 금융 지수 정보 노출
9. 트레져러 이야기
    - 클릭 시 [블로그] 화면으로 이동
10. 공지사항
    - 클릭 시 [공지사항] 화면으로 이동

# Feed

- 파일명: Feed
- 버전: v.1.16.3
- 화면설명: 오늘의 시장분석 및 금융 콘텐츠를 빠르게 탐색할 수 있는 피드 화면
1. 피드 콘텐츠
    - 오늘의 종목
    - 시장 심리
    - 뉴스
    - 퀴즈
    - 기타 금융/경제 콘텐츠
2. 피드 탐색
    - 위/아래 스와이프를 통해 다음 콘텐츠 탐색 가능
    - 좌/우 스와이프를 통해 동일 콘텐츠의 추가 페이지 확인 가능
3. 사용자 반응
    - 콘텐츠에 대한 좋아요 / 싫어요 평가 가능

# AlphaChat

- 파일명: AlphaChat
- 버전: v.1.16.3
- 화면설명: 주식, 원자재 및 금융 시장 관련 질문에 답변해주는 AI 알파 에이전트 화면
1. 추천 질문
    - 사용자 진입 시 추천 질문 6개 예시 노출
    - 추천 질문 클릭 시 자동으로 질문 입력 및 답변 생성
2. 채팅 기능
    - 사용자가 직접 질문 입력 가능
    - 입력된 질문에 대해 알파챗이 AI 기반 답변 제공
3. 제공 콘텐츠
    - 주식
    - 원자재
    - 시장 분석
    - 경제/금융 정보

# Benefit

- 파일명: Benefit
- 버전: v.1.16.3
- 화면설명: 다양한 참여형 미션을 통해 리워드를 획득할 수 있는 혜택 화면
1. 내 리워드
    - 현재 보유 중인 골드티켓 및 포인트 금액 확인 가능
2. 포인트 전환 미션
    - 골드티켓을 사용하여 참여 가능한 포인트 전환 미션 제공
3. 미션 콘텐츠
    - 경제 레슨
    - 예측 대회
        - 예측 참여에 사용 가능한 가상 재화 ‘Pick’ 제공
        - 시즌 종료 후 순위에 따라 우승 상금 정산
    - 출석체크
    - 걸음 수 미션
    - 기타 리워드형 참여 미션
4. 리워드 시스템
    - 미션 완료 시 골드티켓 및 포인트 획득

    - 보유 중인 종목 및 평가금액 확인 가능

# Market

- 버전: v.1.16.3
- 화면설명: 원자재, 명품 및 다양한 조각 투자 상품을 탐색할 수 있는 마켓 화면

    - 상품 클릭 시 상세 정보 및 거래 화면으로 이동
