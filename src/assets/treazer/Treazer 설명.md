# Treazer 설명

# Home

- 파일명: Home
- 버전: v.1.3.7
- 화면설명: 홈 메인 화면
1. 내 금포인트
    - 현재 보유 중인 금포인트 노출
    - 클릭 시 [히스토리] 화면으로 이동
2. 오늘의 출석퀴즈
    - 출석퀴즈 풀이 완료 시 자동으로 출석 체크 처리
    - 체크인 완료 시 버튼은 ‘골드 더 벌기’ 상태로 변경됨
        - 골드 더 벌기 버튼 클릭 시 → [경제 퀴즈] 화면으로 이동
3. 이상형 월드컵
    - 부스팅 콘텐츠 영역
    - 카드 형태로 리스트 노출
    - 카드 클릭 시 [이상형 월드컵] 화면으로 이동
    - 더보기 클릭 시 [이상형 월드컵 리스트] 화면으로 이동
    - 이상형 월드컵은 운영 배너형 콘텐츠로 관리
    - 현재 베트남(VN) 국가에서만 노출
        - 일본(JP) / 태국(TH) / 싱가폴(SG) 버전에서는 비노출 처리
4. 경제 퀴즈
    - 랜덤 퀴즈 2개 노출
    - 사용자가 모든 퀴즈를 완료하기 전까지 유지
    - 더보기 클릭 시 [경제 퀴즈] 화면으로 이동
5. 데일리 퀴즈
    - 뉴스/기사 기반 AI 생성 퀴즈
    - 노출 개수 미정
    - 카드 클릭 시 [퀴즈 상세] 화면으로 이동

# **Mission**

- **파일명: Mission**
- **버전: v.1.3.7**
- **화면설명: 다양한 미션 콘텐츠를 모아볼 수 있는 미션 메인 화면**
1. 오늘의 출석퀴즈
    - 홈 화면과 동일한 정책 적용
    - 출석퀴즈 풀이 완료 시 자동으로 출석 체크 처리
    - 체크인 완료 시 버튼은 ‘골드 더 벌기’ 상태로 변경됨
        - 골드 더 벌기 버튼 클릭 시 → [경제 퀴즈] 화면으로 이동
2. 데일리 미션
    - 매일 초기화되며 재참여 가능
    - 다양한 참여형 미션 콘텐츠 제공
    - 미션 종류는 추후 운영 정책에 따라 변경 및 추가 가능
3. 시간제 미션
    - 정해진 시간대에 따라 미션 오픈
    - 종료된 미션은 다음날 동일 시간대에 다시 오픈

- 미션종류: 퀴즈 1유닛(5문제 정답) 완성하기
- 미션 시간대:
    - 아침 미션 (오전 9:00 ~ 오전 11:59)
    - 점심 미션 (오후 12:00 ~ 오후 2:59)
    - 오후 미션 (오후 3:00 ~ 오후 6:59)
    - 저녁 미션 (오후 7:00 ~ 오후 11:59)

- 오픈된 미션 클릭 시 [퀴즈 풀기] 화면으로 랜덤 이동
- 미션 완료 시 ‘보상 받기' 버튼 활성화
- ‘보상 받기’ 클릭 시 리워디드 광고(30초) 노출 후 리워드 지급

# Quiz

- 파일명: Quiz
- 버전: v.1.3.7
- 화면설명: 경제/금융 퀴즈를 풀이할 수 있는 퀴즈 화면
1. 퀴즈 방식
    - 4개의 보기 중 정답 1개를 선택하는 객관식 퀴즈 방식으로 구성
2. 퀴즈 보상
    - 정답 선택 시 10골드 지급
    - 리워드 단위 기준
        - 1 GOLD = 0.0000005 g (약 0.1원)
3. 정답 및 해설
    - 정답 확인 시 해설 콘텐츠 노출
    - 문제에 대한 추가 설명 및 학습 정보 제공
4. 유닛 시스템
    - 누적 정답 5개 달성 시 1유닛 완성 처리

# Store

- 파일명: Store
- 버전: v.1.3.7
- 화면설명: 금포인트를 사용하여 다양한 상품을 교환할 수 있는 스토어 메인 화면
1. 스토어 메인
    - 금포인트로 교환 가능한 상품 목록 제공
    - 국가별 스토어 상품이 다르게 노출됨
    - 일본(JP) 버전의 경우 기프티(giftee) 상품 노출
2. 내 금포인트
    - 현재 보유 중인 금포인트 노출
    - 더보기 아이콘 클릭 시 [금 시세] 화면으로 이동
3. 내 쿠폰함
    - 구매한 쿠폰 확인 가능
    - 클릭 시 [내 쿠폰] 화면으로 이동
4. 광고 배너
    - 네이티브 광고 or 디스플레이 광고
5. 상품 카테고리
    - 총 9개 카테고리 구성 - Beauty & Health - Cafe & Beverage - Entertainment - Fashion - Food & Dining - Grocery - Lifestyle - Others - Voucher
6. 상품 목록
    - 선택된 카테고리에 따라 상품 목록 변경
    - 상품 가격은 금 시세 기준으로 실시간 변동
    - 상품 클릭 시 [상품 상세] 화면으로 이동
